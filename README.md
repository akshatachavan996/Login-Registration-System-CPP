# 🔐 Login and Registration System (C++)

A simple console-based Login and Registration System developed in **C++**. The application allows users to register, log in, validates user input, checks for duplicate usernames, and stores user credentials using file handling.

---

## ✨ Features

- 👤 User Registration
- 🔑 User Login
- ✅ Username and Password Validation
- 🚫 Duplicate Username Detection
- 💾 File Handling for Credential Storage
- 📢 Success and Error Messages
- 📋 Menu-Driven Console Interface

---

## 🛠️ Technologies Used

- C++
- Functions
- File Handling
- Strings
- Conditional Statements
- Loops

## 📖 How It Works

1. Register with a unique username and password.
2. The program validates the entered details.
3. User credentials are stored in `users.txt`.
4. Log in using the registered username and password.
5. The program displays appropriate success or error messages.
